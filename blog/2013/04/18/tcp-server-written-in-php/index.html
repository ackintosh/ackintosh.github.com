PHPでTCPサーバー

Apr 18, 2013 #php

個人的には、pcntl_fork()したあとswitch文で分岐する流れが、
理解するのに苦労しました。

ちょうどRubyで並列処理を勉強していたのですが、
やっぱりRubyの方が直感的で書きやすいですね。
