MySQL(InnoDB)でバルクinsertのベンチマークとってみました

by Ackintosh - Jul 18th, 2012 - posted in MySQL

テーブル作成

create table test (
id int(11) unsigned auto_increment primary key,
name varchar(255) not null,
created datetime not null
);

通常のinsert文

insert into test (name, created) values ('name1', now());
insert into test (name, created) values ('name2', now());
・・・ 10,000回

結果
6.39秒

バルクinsert

insert into test (name, created) values ('name1', now()), ('name2', now()) ・・・
・・・10,000回

結果
0.20秒

全然違う(・∀・)
